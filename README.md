# Obelisk VPN

VPN client with L7-Shred protocol implementation.

## Supported Platforms

- Windows
- Android

## Latest Releases

All builds and binary assets are available on the [Releases Page](https://github.com/pioneer-dj/obelisk-vpn/releases).

---

## Installation & Usage

### Windows

1. **Download** `obelisk-vpn-v1.0.0-windows-x64.zip` from the releases page.
2. **Extract** the entire archive to a separate folder.
3. **Run** `ObeliskVPN.exe` directly from that folder.

> **Important:** All files extracted from the archive must remain in the same directory. Do not move the `.exe` file out of its folder.

### Android

1. **Download** `obelisk-vpn-v1.0.0-android-arm64.apk` from the releases page.
2. **Install** the APK file.
3. **Open** Obelisk VPN from your app drawer.
